# KeypadInput

Bibliotek som demonstrerer hvordan man leser matrise-tastatur ved hjelp av Keypad-biblioteket. Klassen arver fra `BaseSensor` og logger tastetrykk på norsk.

## Maskinvare

- 4x4 matrise-tastatur koblet til digitale pinner (rader 2–5, kolonner 6–8 og A1 i eksempelet)

## Programvare

Installer `Keypad`-biblioteket i Arduino IDE.

## Eksempel

Se `examples/Keypad_AccessControl` for kombinert bruk med RFID.
