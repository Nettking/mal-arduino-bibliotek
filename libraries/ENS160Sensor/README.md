# ENS160Sensor

Bibliotek som kapsler Sensirion ENS160 gass-sensoren og viser hvordan vi kan lese luftkvalitetsindeks og estimert CO2-ekvivalent.

## Maskinvare

- ENS160-modul koblet til I2C (adresse 0x52)

## Programvare

Installer `ENS160`-biblioteket (SparkFun eller tilsvarende) i Arduino IDE.

## Eksempel

Se `examples/ENS160Sensor_Example`.
