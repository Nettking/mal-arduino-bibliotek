# LDRSensor

Bibliotek for enkel avlesing av en lysavhengig motstand (LDR). Klassen arver fra `BaseSensor` og leverer `begin()` og `read()` implementasjoner med norsk logging.

## Maskinvare

- LDR koblet i spenningsdeler mot 5 V
- Midtpunkt tilkoblet analog inngang (standard A0)

## Eksempel

Se `examples/LDRSensor_Example` for en komplett skisse.
